## 2. COLLECTION TYPE: Proyecto

Crea el Content Type con exactamente estos campos:

| Campo        | Tipo en Strapi     | Configuración                          |
|--------------|--------------------|----------------------------------------|
| id           | Auto (Strapi)      | Generado automáticamente               |
| titulo       | Short Text         | Required, MaxLength: 150               |
| descripcion  | Long Text          | Required                               |
| stack_sluck  | JSON               | Lista de tecnologías del proyecto      |
| image        | Media (Single)     | Formatos: jpg, jpeg, png, webp, svg    |
| link_demo    | Text               | URL del proyecto en producción         |
| link_github  | Text               | URL del repositorio en GitHub          |

## 4. PERMISOS DE LA API

Configura los permisos en Settings → Roles:

### Public (sin token):
✅ GET  /api/proyectos          → find (listar todos)
✅ GET  /api/proyectos/:id      → findOne (detalle)
❌ POST, PUT, DELETE            → solo admin

### Authenticated (con token):
✅ Todos los métodos CRUD

## 5. ENDPOINTS DISPONIBLES

Una vez configurado, estos son los endpoints:

// Listar todos los proyectos
GET http://localhost:1337/api/proyectos

// Obtener un proyecto por ID
GET http://localhost:1337/api/proyectos/:id

// Con imagen incluida (populate)
GET http://localhost:1337/api/proyectos?populate=image

// Todos los campos incluyendo imagen
GET http://localhost:1337/api/proyectos?populate=*

// Crear proyecto (requiere autenticación)
POST http://localhost:1337/api/proyectos
Content-Type: application/json
{
  "data": {
    "titulo": "Mi Portfolio",
    "descripcion": "Portfolio personal con React y MUI",
    "stack_sluck": ["React", "TypeScript", "Material UI", "Vite"],

    "link_github": "https://github.com/usuario/portfolio"

// Actualizar proyecto (requiere autenticación)
PUT http://localhost:1337/api/proyectos/:id

// Eliminar proyecto (requiere autenticación)

## 7. TIPOS TYPESCRIPT PARA EL FRONTEND

Dame los tipos listos para usar en React:

// types/proyecto.types.ts

export interface ImageFormat {
  url: string
  width: number
  height: number
}

export interface ProyectoImage {
  id: number
  name: string
  url: string
  width: number
  height: number
  formats: {
    thumbnail?: ImageFormat
    small?: ImageFormat
    medium?: ImageFormat
    large?: ImageFormat
  }
}

export interface Proyecto {
  id: number
  documentId: string
  titulo: string
  descripcion: string
  stack_sluck: string[]        // JSON array de tecnologías
  image: ProyectoImage | null
  link_demo: string | null
  link_github: string | null
  createdAt: string
  updatedAt: string
  publishedAt: string | null
}

export interface StrapiResponse<T> {
  data: T[]
  meta: {
    pagination: {
      page: number
      pageSize: number
      pageCount: number
      total: number
    }
  }
}

export type ProyectosResponse = StrapiResponse<Proyecto>

## 8. SERVICE EN REACT (Axios)

Dame el service completo para consumir desde el frontend:

// features/projects/services/proyectoService.ts

import api from '@/lib/api'
import type { ProyectosResponse, Proyecto } from '@/types/proyecto.types'

// Obtener todos los proyectos con imagen
export const getProyectos = async (): Promise<Proyecto[]> => {
  const { data } = await api.get<ProyectosResponse>(
    '/proyectos?populate=image'
  )
  return data.data
}

// Obtener un proyecto por ID
export const getProyectoById = async (id: number): Promise<Proyecto> => {
  const { data } = await api.get<{ data: Proyecto }>(
    `/proyectos/${id}?populate=image`
  )
  return data.data
}

## 9. HOOK CON TANSTACK QUERY

// features/projects/hooks/useProyectos.ts

import { useQuery } from '@tanstack/react-query'
import { getProyectos, getProyectoById } from '../services/proyectoService'

// Lista de proyectos
export const useProyectos = () => {
  return useQuery({
    queryKey: ['proyectos'],
    queryFn: getProyectos,
    staleTime: 1000 * 60 * 5,   // 5 minutos de caché
  })
}

// Un proyecto por ID
export const useProyecto = (id: number) => {
  return useQuery({
    queryKey: ['proyecto', id],
    queryFn: () => getProyectoById(id),
    enabled: !!id,
  })
}
